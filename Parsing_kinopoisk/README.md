# Документация по программе скрапинга данных о фильмах и рейтингах с Кинопоиска
## Программа предназначена для скрапинга данных о фильмах и их рейтингах с веб-сайта Кинопоиск. 

### Она выполняет следующие задачи:

1. Скрапинг данных о фильмах с кассовыми сборами:
    - Используя API-запрос к GraphQL-серверу Кинопоиска, программа получает информацию о фильмах с кассовыми сборами за определенный период времени. 
    - Из полученных данных извлекаются названия фильмов, бюджеты и данные о кассовых сборах в России, США и мире. 
    - Данные сохраняются в CSV-файл 'movies_data.csv' с указанием названия и бюджета каждого фильма.
2. Скрапинг рейтингов фильмов:
- Используя Selenium WebDriver, программа запускает веб-браузер Chrome в режиме без графического интерфейса. 
- Для каждого фильма в списке полученных ранее, программа открывает страницу фильма на Кинопоиске. 
- С помощью BeautifulSoup, программа находит на странице рейтинг фильма и извлекает его значение. 
- Рейтинг каждого фильма добавляется в соответствующую строку CSV-файла 'movies_data.csv'. 

### Использование программы:

1. Установите необходимые зависимости:
- requests 
- beautifulsoup4 
- selenium
2. Установите Chrome WebDriver и укажите путь к нему в коде программы:
- python 
- Copy code 
- service = Service('/path/to/chromedriver')
3. Запустите программу, выполнив следующий код:

`__if __name__ == '__main__'__ `

4. После выполнения программы будет создан файл 'movies_data.csv', содержащий данные о фильмах и их рейтингах.
